Place your profile image here so it appears in the portfolio:

- File path: `assets/avatar.jpg`
- Recommended size: 500×500 px (square); the CSS will crop to a circle.
- Acceptable formats: `.jpg`, `.jpeg`, `.png`.

If you upload the image with the exact filename above, it will automatically be displayed on the homepage. If the file is missing or invalid, a placeholder SVG will be shown instead.
